# El Dorado SEO Engine

The SEO Engine automates everything required to give generated websites premium ranking power immediately.

## Purpose

This microservice handles:

### 1. **Auto Metadata**
Automatically generates:
- Title tags
- Meta descriptions
- OG tags
- Twitter cards
- Schema.org structured data

### 2. **Auto Page Structuring**
AI-powered page optimization:
- Heading structure (H1/H2/H3)
- Keyword placement
- Readability scoring
- Image alt tags
- Internal linking suggestions

### 3. **Auto Indexing**
- Trigger search indexing for Google & Bing
- Manage IndexNow submissions
- Verify sitemap URLs

### 4. **Sitemap Submission**
- Auto-generate sitemaps
- Push them to Google Search Console
- Monitor indexing status

## Architecture
- Node.js TypeScript microservice
- Handles batch SEO scoring queues
- Communicates with backend via messages or REST events

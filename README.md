# El Dorado
### Turning your ideas into gold.

El Dorado is an AI-powered website generator that transforms spoken business ideas into revenue-ready websites. Users speak their vision, El Dorado completes the 7-step business success framework, generates a full website blueprint, builds a site using V0, and deploys it to Vercel with domains + SEO handled automatically.

## Core Features
- Speech-to-website generation
- 7-step business framework engine
- Conversational site editing (vibe coding)
- V0-powered UI generation
- One-click deploy to Vercel
- Domain search + purchase
- Automated SEO + indexing
- Fully revenue-ready websites
